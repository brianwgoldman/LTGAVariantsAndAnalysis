Linkage Tree Genetic Algorithms: Variants and Analysis
==================================================================================

These pages provide code documentation for the Linkage Tree Genetic Algorithms: Variants and Analysis paper published in the
Genetic and Evolutionary Computation Conference 2012 by Brian W. Goldman and Daniel R. Tauritz.  For information about how
to run this code, please see the :mod:`ltga.main` module.  For support questions feel free to email brianwgoldman@acm.org or 
dtauritz@acm.org.
